# nest-react-query
nest js and react query application project

- Backend
    - goto server_t folder
    - npm install
    - npx prisma generate
        - npx prisma studio [for database UI editor]
    - npm run start:dev
    - http://127.0.0.1:3500/api [for swagger api documentation]

- Frontend
    - goto client folder
    - npm install
    - npm run dev
    - http://127.0.0.1:5173 [for UI]
